How to configure IceRaven for Android:

Configuration via Webextension

  1. Open IceRaven and click the main menu. Scroll to the top, and tap the "Add-Ons" submenu.
  2. Tap on the "Add-Ons Manager" option.
  3. Find the "I2P Proxy for Android and Other" extension in the Add-Ons collection and tap on the + symbol
  4. You can now browse I2P Sites!
